TAXONOMÍA

LLM, Copilot y Agente.

Tres categorías que parecen lo mismo y no lo son.

Tipo Qué hace Ejemplos
LLM Responde a prompts de forma conversacional. ChatGPT · Claude · Gemini
Copilot Sugiere código en tiempo real, dentro del editor. GitHub Copilot

Cada categoría exige un nivel distinto de criterio del dev.

22 / 24 · INFRAESTRUCTURA
MCP
El protocolo que conecta todo

MCP: el USB-C de la IA.

Qué es

Model Context Protocol — un estándar abierto para que cualquier modelo se conecte a herramientas, datos y servicios externos.

Por qué importa

Lo escribís una vez y funciona en Claude, Cursor, VS Code, cualquier cliente compatible.

Antes vs ahora
×
Cada IA necesitaba su integración propia
×
Código custom por cada herramienta
✓
Un server MCP → todos los clientes
Ejemplos reales
Base de datos
Slack
CI/CD
Jira
APIs internas
La frase

El dev que sabe armar un MCP server le da superpoderes a toda su organización.
